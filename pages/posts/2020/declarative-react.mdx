In this post I will break down some common mistakes/inefficiencies I see in react components, and how you can refactor them as clean and [declarative]

## Tip #1. Destructure props as early as possible

Unless you need to pass the entire "props" object somewhere, [destructure] your props to eliminate repeating "`props.`" throughout the component

_Instead of this_

```jsx 1
-  const Component = props => {
-   return (
+    <div>
+      <h1>{props.title}</h1>
      <span>{props.date}</span>
      <p>{props.description}</p>
    </div>
  );
};
```

_Do this_

```jsx
const Component = ({ title, date, description }) => {
  return (
    <div>
      <h1>{props.title}</h1>
      <span>{props.date}</span>
      <p>{props.description}</p>
    </div>
  );
};
```

## Tip #2. Sets prop defaults

```jsx

const Component: React.FunctionComponent = (props) => {
    var propertyA = "property default"; // set a default for property A
    if (
        props.property_a !== null &&
        props.property_a !== undefined &&
        props.property_a !== ""
    ) {
        profile_url = props.profile_url;
    }
```
